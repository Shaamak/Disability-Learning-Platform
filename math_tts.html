Maths Basics

Solve for x in linear equations and simplify expressions. Equation example: 2x + 3 = 7 Solution: x = 2
Standard form: ax^2 + bx + c = 0 Solutions are given by the quadratic formula: x = (-b ± √(b^2 - 4ac)) / 2a
Common trigonometric ratios: sin(θ) = opposite / hypotenuse cos(θ) = adjacent / hypotenuse tan(θ) = opposite / adjacent
Pythagorean identity: sin^2(θ) + cos^2(θ) = 1
Derivatives and Integrals are the foundations of calculus. Derivative example: If f(x) = x^2, then f'(x) = 2x Integral example: ∫ x dx = (x^2 / 2) + C
Matrix addition, subtraction, and multiplication. Matrix multiplication: If A is an m × n matrix and B is an n × p matrix, the resulting matrix C is m × p.
Basic concepts: P(A) = Number of favorable outcomes / Total number of outcomes P(A ∩ B) = P(A) * P(B) for independent events
Mean: Sum of all data points / Number of data points Median: Middle value of an ordered data set Mode: Most frequently occurring value in a data set Standard deviation measures the dispersion of a dataset relative to its mean.
Properties of shapes such as triangles, circles, and polygons. Area of a circle: A = πr^2 Circumference of a circle: C = 2πr
